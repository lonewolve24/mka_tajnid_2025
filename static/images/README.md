# Logo File

Please add your logo file here as `logo.png`

The logo should be:
- PNG format (or JPG/SVG - update template if using different format)
- Recommended size: 200x200px or larger (will be scaled automatically)
- Circular or square format works best

After adding the logo file, the system will automatically display it:
- On the login page (replacing the text heading)
- In the navbar (alongside the text, not blocking it)
